# Kai Family Album

Interactive family album portfolio site.

## Local Preview

```bash
npm run dev
```

Open `http://127.0.0.1:5176`.

## Vercel Deployment

This repository is configured for Vercel static output.

- Build command: `npm run build`
- Output directory: `dist`
- Production files are copied by `scripts/build-vercel.mjs`

After the GitHub repository is imported into Vercel, every push to `main` will trigger a new deployment automatically.
